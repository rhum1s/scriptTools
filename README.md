# Script Tools
Small tools to help computing scripts creation.
    
## PgDb  
Connect and use PostgreSQL / PostGIS with Python.  
  
###  Use  
Create a configuration file.    
Instanciate a PgDb object as demonstrated in pgdb.py.    
  
### Config file example   
[Connection]  
host = xxx.xx.xx.xxx  
lgn = xxx  
pwd = xxx
bdd = xxx  
port = 5432
